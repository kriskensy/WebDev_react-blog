# Educational Blog Posts Application

This project is an educational demonstration of building a blog post platform using React and Redux. It is intended to help developers learn state management, routing, and component design in a modern React environment.

## Key Features

- **React and Redux Integration**: Demonstrates managing posts and categories within Redux store.
- **Dynamic Routing**: Utilizes React Router to navigate between all posts, category-specific post listings, and single post views.
- **Filtering Posts by Category**: Shows how to filter and display posts dynamically based on category selected.
- **Component Architecture**: Illustrates separation of concerns with reusable components like PostCard and CategoryPosts.
- **CSS Grid Layout**: Implements responsive layouts using CSS Modules and Grid.
- **Error Handling**: Covers basic UI feedback for empty lists or categories with no posts.

## Educational Focus

This app serves as a learning tool for:

- State initialization and updates via Redux reducers and selectors.
- Safe and effective use of React hooks like `useSelector` and `useParams`.
- Best practices in routing with parameters for dynamic content.
- Debugging and handling edge cases such as missing data or URLs.
- UI layout management combining Bootstrap and CSS Modules.

## Getting Started

- Clone the repository  
- Install dependencies with `npm install` or `yarn`  
- Run locally via `npm start` or `yarn start`  
- Explore and modify source code to deepen understanding
